# Изображения для примеров

Поместите сюда изображения для секции "Примеры использования" на лендинге.

## Как использовать:

1. **Загрузите изображения** в эту папку (`frontend/public/images/examples/`)
2. **Используйте путь** в формате: `/images/examples/your-image.jpg`
3. **Обновите** `frontend/app/data/landingData.ts`:

```typescript
{
  title: "Лендинг для бизнеса",
  image: "/images/examples/business-landing.jpg", // вместо внешней ссылки
}
```

## Рекомендации:
- Формат: JPG или PNG
- Размер: 800x600px или больше (соотношение 16:9 или 4:3)
- Оптимизируйте изображения перед загрузкой (используйте TinyPNG или аналоги)

## Альтернативные сервисы для загрузки:

1. **ImgBB** (https://imgbb.com) - уже используете
2. **Cloudinary** (https://cloudinary.com) - с оптимизацией
3. **GitHub** - загрузите в репозиторий и используйте raw ссылку
